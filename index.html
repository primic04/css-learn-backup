ou ecrit-on le css?

on l'ecrit dans un fichier spécial ayant pour extension .css. C'est la méthode la plus pratique et la plus souple

il est inutile d'ouvrir le fichier .css dans le navigateur, il faut toujours ouvrir le fichier avec l'extansion .html, il fera automatiquement appel au fichier style.css avec la ligne de code

( link rel="stylesheet" href="style.css" )

appliquer un style: sélectionner une balise

dans un code CSS3 on trouve trois éléments différents

  1. des noms de balises
    • on ecrit le nom de la balise que l'on souhaite modifié l'apparence

      Ex: la balise(< p >) pour la modifier enlever les < > il ne reste que les lettre de la balise

  2. des propriété CSS
    • "les effets de style" de la page sont rangé dans des propriétées

      Ex: la propriété color permet d'indiquer la couleurs du texte, font-size précise la taille du texte etc ...

  3. des valeurs
    • pour chaque propriété CSS on doit indiquer une valeur.

      Ex: la propriété "color", il faut indiquer le nom de la couleur en valeur; pour la propriété font-size , on indique la taille que l'on souhaite en valeur

EX:


balise 1

{

     propriété1 : valeur1

     propriété2 : valeur2

}

balise 2

{

     propriété1 : valeur1

     propriété2 : valeur2

     propriété3 : valeur3

     propriété4 : valeur4

}

balise 3

{

     propriété1 : valeur1

     propriété2 : valeur2

     propriété3 : valeur3

}

     propriété1 : valeur1

     propriété2 : valeur2

     propriété3 : valeur3

     propriété4 : valeur4

En résumé

  1. on écrit le nom de la balise sans les < >
  2. on ouvre des accolades { }
  3. à l'interieur, ajouter les propriétées
  4. suivi de la propriété on met les : (deux point)
  5. suite au deux points on entre la valeur de la propriété qu'on veut

    6. chacunes des propriétées est suivis du symbole ":" puis de sa valeur et se termine par des points virgule ";"

    on peut mettre entre les accolade autant de propriété suivis de sa valeur que l'on veux: PAS DE MAXIMUM

Appliquer un style: class et id

Ces attributs speciaux fonctionnent sur toutes les balises. ces deux attributs sont quasiment identiques.

  1. l'attribut class

      2. l'attribut class peut etre associé a n'importe quelle balise, image, paragraphe, titre, etc ...

    • lorsque vous ajouter l'attribut class a votre balise, c'est fait votre balise est identifier au nom que vous lui avez attribué

      • et alors vous pourrez reutilisé le nom attribué dans le fichier .css pour l'appeler

  2. l'attribut id

      3. l'attribut id fonctionne exactemment de la même maniere que class

    • En revenche il ne peux être attribué qu'une seule fois dans tous le code

      • très pratique pour faire un ancre sur un logo. Nous n'utiliserons l'id que sur des éléments uniques dans la page

      si vous utilisé l'id, lorsque vous definirez ses propriétées dans le fichier .css, il faudra précèder son nom par un (#)

les balises Universelles

en effet le probleme de l'attribut class , c'est qu'il s'agit d'un attribut donc, que vous ne pouvez en mettre que dans les balises

en realité elle sont dites universelle puisqu'elle n'ont aucune signification particulière. elles sont en revanche très importante

  1. < span >:
    • c'est une balise de type "inline", c'est a dire qu'on la place au sein d'une paragraphe de texte, pour sélectionner certain mot uniquement
  2. < div >:
    • c'est une balise de type "block", qui entoure un bloc de texte, comme:
      < p > et < h1 >, etc ... elle créé automatiquement un retour a la ligne.

      • div est tres frequemment utilisé dans la construction d'un design
